### [Model Version History & Limitations]
1. Model v1 (Mask R-CNN): 
   - 경위: 초기 DENTEX 평가용. 
   - 한계: 2-stage 구조라 속도가 느림(18 FPS), mAP가 낮고 Recall이 낮음(14%).
2. Model v2 (YOLOv8m-seg, `yolov8m_seg_run`): 
   - 경위: 실시간성 및 성능 확보를 위해 1-Stage 모델로 마이그레이션. 
   - 한계: 속도(46 FPS)와 Recall(42%)은 비약적으로 상승했으나, 턱뼈/임플란트를 치아로 착각하는 오탐지(FP: 1174)가 극심함.
3. Model v3-Crop (YOLOv8m-seg, `yolov8m_seg_finetune`):
   - 경위: 외부 데이터 부재로 기존 이미지의 빈 공간을 크롭해 500장 배경 데이터로 투입 후 파인튜닝.
   - 한계: 부분적인 패치 이미지만으로는 턱뼈의 해부학적 맥락을 학습하지 못해 FP 감소에 실패함(FP: 1180).
4. Model v4-HardNegative (진행 예정):
   - 경위: 사용자가 직접 수집한 75장의 실제 무치악/임플란트 픽스처 엑스레이를 배경 데이터(빈 라벨)로 병합하여 '처음부터 다시 학습(Train from scratch)' 진행. 기존 가중치의 편향을 초기화하고 강건한 네거티브 특징을 학습할 예정.
